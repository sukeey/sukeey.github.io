使用hugo搭建个人博客站点

Written by sukeey with ♥ on 26 April 2019 in 2 min

使用hugo搭建个人博客站点

1.

Hugo二进制下载地址戳这里,解压后会有一个叫hugo.exe或者hugo的程序。

2.

生成自己的站点 用命令行方式

$ hugo new site mysite

那么在mysite里面会自动生成以下的目录：

archetypes/

content/

layouts/

static/

themes/

config.toml

注: congig.toml是网站的配置文件(有时你要用一个主题，可能会改动这个配置文件),content目录里放的是你写的markdown文章，layouts目录里放的是网站的模板文件，static目录里放的是一些图片、css、js等资源。

3.

进入生成的site目录

$ cd mysite 
#创建一个页面
$ hugo new about.md
#若是博客日志，最好将其放在content的post下
$ hugo new post/first.md
#这样会在content/post下生成一个MarkDown格式的first.md文件

4.

接下来就可以写文章,draft 表示是否是草稿，编辑完成后请将其改为 false，否则编译会跳过草稿文件。 附：MarkDown基本语法

5.

有一点，默认生成的网站是没有任何皮肤的，但没有主题又无法工作，所以要去主题网站下载一个主题到themes中.

$ cd themes
$ git clone https://github.com/orianna-zzo/AllinOne.git themes/AllinOne
$ tree    # 检测主题是否下载成功，一般可以直接看出来。

(当然网好的话可以下载所有的主题)

6.

进行本地调试

$ hugo server --theme=AllinOne --buildDrafts --watch

解释： 在项目根目录下(mysite目录下)，通过 hugo server 命令可以使用 hugo 内置服务器调试预览博客。–theme 选项可以指定主题，–watch 选项可以在修改文件后自动刷新浏览器，–buildDrafts 包括标记为草稿(draft)的内容。

部署网站到github page

首先要在GitHub上创建一个Repository，命名为：sukeey.github.io （sukeey替换为你的github用户名）。 在站点根目录(mysite)执行 Hugo 命令生成最终页面：

$hugo

可看到根目录(mysite)下生成/public文件夹，public的内容即Hugo生成的整个静态网站。将pubilc目录里所有文件 push 到刚创建的Repository的 master 分支。在public目录下右击打开git bush 过程如下：

$ git init     本地库初始化
$ git remote add origin https://github.com/sukeey/sukeey.github.io.git      添加一个新的remote repo
$ git add -A    一次添加所有改动的文件
$ git commit -m "first commit"     将暂存区的内容提交到本地库
$ git push -f origin master

浏览器里访问：http://sukeey.github.io/ 即可

过程中可能出现的问题

fatal: remote origin already exists.

表明这个origin库已经存在，那么删掉 git remote rm origin 库，重新执行该命令即可。

git push时报错The requested URL returned error: 403

在网上搜到要更改配置文件，但我改了还是没用，后看到一篇文章原文链接 中的命令：

$rundll32.exe keymgr.dll,KRShowKeyMgr

执行这条语句后会弹出一个窗口，删掉里面的账号信息，重新git push 会弹窗让你输入用户名及密码，最后执行git push -f origin master 成功。

输入 git push origin master 报错

$ error: failed to push some refs to'https://sukeey@github.com/sukeey/sukeey.github.io.git

原因是远程库与本地库不一致，在hint中有提示要把远程库同步到本地库，执行以下命令即可：

$ git pull --rebase origin master

使用自己的域名

也可以使用自己的域名，购买域名的地方很多，如网站 dnspod.cn。购买完成后，在dnsod.cn 的域名解析中添加记录。在上面存放静态网站的Repository设置里面Custom domain填上自己的域名点击save即可 这样就可以使用 sukeey.com 来访问了。
